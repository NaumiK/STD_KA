# STD_KA
Модель для диаграммы переходов состояний (модель автомата по продаже горячих напитков (чай, кофе,…) с оплатой через наличный расчет (принимаются купюры 500. 100р, 50р., монеты 10р., 5р., 2р., 1р.). Сдача – наличными)
